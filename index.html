🏆 NBA Finals Media Delivery Platform

Cloud Engineering Project • AWS • Linux • Git • CloudFront

Delivering Knicks vs Spurs Championship Content Worldwide

Millions of basketball fans are refreshing NBA.com every few seconds after Game 7. They expect instant access to:

  • 🏀 Game Highlights
  • 📸 Championship Photos
  • 🎤 Player Interviews
  • 📰 Breaking News
  • 📊 Box Scores

🚨 The Problem

After Game 7 ends... Over 35 million fans try accessing the website at exactly the same time. Without a Content Delivery Network (CDN), the website becomes slow, images fail to load, and videos constantly buffer. As the Cloud Engineer, your job is to keep the website online during the biggest basketball event of the year.

☁️ My AWS Architecture

  • Amazon S3 stores all Finals photos and videos.
  • Amazon CloudFront delivers content from edge locations worldwide.
  • GitHub stores source code.
  • VS Code is used for development.
  • Linux Terminal manages deployments.

🏀 Basketball Analogy

CloudFront is like having Steph Curry warm up in every NBA arena. Instead of flying Steph from San Francisco every game... there's already a Steph waiting nearby. Likewise... CloudFront keeps copies of your website around the world so fans receive content from the closest server. That means faster highlights and happier fans.
